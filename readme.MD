## âš™ï¸ **Configuration**

Edit `config.js` or `.env` file with your details:

```javascript
module.exports = {
  OWNER_NUMBER: "94xxxxxxxxx", // Your WhatsApp number
  BOT_NAME: "SHANU-MD",
  PREFIX: ".",
  MODE: "public", // or "private"
  AUTO_READ: true,
  AUTO_STATUS_READ: false,
  PRESENCE: "available" // online, typing, recording
};
```
